Viregule

PHP, MySQL, JavaScript, jQuery, Ajax, HTML, CSS, Bootstrap

Viregule is a web application developed for the French National Railway Corporation (SNCF). I designed and developed this whole project autonomously from discussions and regular meetings with end users. The purpose of the application is to follow the investigations on the origins of the train delays. This web application is for internal use, it is therefore not accessible by everyone.

Here is the home page that allows the user to see the number of delays to be treated. The user can also search for a delay in particular if he has the delay number, the delay card then opens.
Here we can see the delays. Each column represents a different status of delay
Here is a delay card. It's a pop-up loaded with Ajax. We can see the work that has already been done on this delay, make changes, or send the delay to different services.
Here is the administration tab which allows to create or modify a user, and to modify other parameters in the application. All actions on this page are managed in Ajax and do not need to reload the page.
