Emaily

React, Redux, Node.js, Express.js, MongoDB, SendGrid, Stripe API

This project is a web application to send surveys to a list of recipients and get the answers in the application. This is a personal project made to improve my React and Node.js skills. Some of the features have been made with the help of the Udemy course "Node with React: Fullstack Web Development".

Here is the home page of the project. The user can login on this page. I use the Passport library with the Google OAuth strategy to allow users to sign in with their Google Account.
Once logged in, the user reaches his list of surveys. He can click "add credits" to buy credits or the "+" button to create a survey.
I use the Stripe API to allow the user to buy credits.
Here is the survey creation form. I use the Redux Form library to manage the data of this form.
Here is the form review page.
Here is an email sent with the platform. I use SendGrid to send the emails and receive the answers.
Back to the survey list, the user can see the answers to his survey.
