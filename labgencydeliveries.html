Labgency Deliveries

PHP, MySQL, JavaScript, jQuery, Ajax, HTML, CSS, Bootstrap

I realized this project largely alone. Labgency is a company that provides VOD services for large customers, as well as web and mobile applications to access these services. Labgency Deliveries is a website allowing customers to access the applications developed for them. The access to this site is restricted to customers and administrators.

Here is the page of deliveries. The deliveries displayed correspond to the OS used. The user can therefore install the application directly from the site. This being the super admin's vision, we can go to another project or create a new one. Each change is loaded with Ajax, so the page is not reloaded.
A delivery may contain additional files to the installation files.
Here is a pop-up for importing a new delivery (loaded with Ajax). Depending on the uploaded files, the site defines the OS of the delivery.
Here are the confirmation, warning or error messages following a delivery.
Here is a pop-up for sending delivery to the customer, accessible from the delieries page.
This is the rights page. The super admin can defines which projects a customer can access, with user or admin rights.
