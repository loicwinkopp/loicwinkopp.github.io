Smart Music Player

Android application

Smart Music Player is a school project created in pairs. It is a random music player that determines the user's musical preferences based on his behavior. For example, if the user zaps the song quickly, the song is unlikely to be replayed. If the user listens to the song until the end, the song will be more likely to be replayed.
